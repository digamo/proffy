## âŒ¨ How To Use

### Clone Repository

```bash
$ git clone https://github.com/digamo/proffy.git
```

### Run API

```bash

# Go to server folder
$ /server

# Install Dependencies
$ npm install

# Create the mapped tables
$ npx knex:migrate

# Run Aplication
$ npm start

```

### Run Web Project

```bash

# Go to web folder
$ /web

# Install Dependencies
$ npm install

# Run Aplication
$ npm start

```
Access url: `http://localhost:3000/`
